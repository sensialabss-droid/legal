## Publicar (GitHub Pages)

1. Repo público (p. ej. `sensialabss-droid/legal`).
2. **Settings → Pages → Build and deployment → Source: Deploy from a branch**, rama `main`, carpeta `/ (root)`.
3. Espera 1–2 min. La web queda en:
   `https://sensialabss-droid.github.io/legal/`

## Pendiente de revisar antes de publicar

- Confirmar el **email de contacto** (`sensialabss@gmail.com`) o cambiarlo por uno de dominio propio.
- Si quieres, añadir **domicilio del responsable** (opcional; hoy se identifica solo con nombre + email).
- Fecha de "Última actualización": `2026-07-05`. Actualízala si editas el contenido.
- Recomendable una revisión legal si escalas (RGPD/LOPDGDD).
